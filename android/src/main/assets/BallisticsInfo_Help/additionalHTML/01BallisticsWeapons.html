BALLISTICS INFO: Pre-mission Weapon Selection
Home Page 2

Prior to first use, users should select and setup any weapon(s) that may be used. Weapon information may be adjusted as required.

Select weapon by accessing the Navigation menu and selecting from the spinner at top.

  1. Weapon Selection Select from Navigation menu.
  2. New adds an additional weapon to the Weapon Selection option. Select this from the options menu on themain screen. Users may create a New weapon or Copy a weapon from previously loaded weapons. Coping a weapon allows the user to adjust information on the weapon.
  3. Edit selected weapon. Weapon information is only editable if using a New weapon; weapons preloaded on the handheld are not editable. Selecting Edit from the options menu makes the Info Field configurable. Info Field contains:
    • Name of weapon.
    • Description.
    • BC (ballistic coefficient) needs to match table based on ballistic model. Default is G1, select G7 for long range.
    • Table is the ballistic model used with BC.
    • Velocity is the muzzle or initial velocity.
    • Sight Height relates to the height of the sighting system above the bore center line.
    • Bullet Weight.
  4. Exit with Back Button. All adjustments are automatically saved continuously.
