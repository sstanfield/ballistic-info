Key information has been highlighted in Red. This screen is used to determine firing solutions. Be sure pre-mission steps 1- Weapons, 2- Zero Information, and 3- Units are completed before using this screen on a mission.

  1. Range is adjustable via sliders of the bottom of the screen. Small increments of range are adjusted on the top slider; large changes of range are adjustable on the bottom slider.
    Changing the Range adjusts output data above.
  2. Bullet Drop/Rise represents the error window of final shot.
  3. Adjust the Unit of Drop by selecting the center unit. Users can adjust using the using the drop icon (upper right).
  4. Wind can be adjusted by selecting on the main screen or through the Navigation menu --> Wind.
  5. Zero distance information is controlled by Zero information. This can be accessed by selecting the Navigation menu -> Zero.
  6. Uphill/Downhill Angle information can be adjusted by selecting Angle or through the Navigation menu. Information is controlled through sliders. Users can adjust if the angle is an Up or Down angle (negative value on the main screen will result from a down angle).
  7. Extra Information displayed is based on current Atmospheric conditions, information on range: velocity, time, and energy of bullet.
  8. Additional Options are available by selecting the Navigation Menu: Atmosphere conditions, Wind information, shooting Angle, Zero information, Unit selection.
