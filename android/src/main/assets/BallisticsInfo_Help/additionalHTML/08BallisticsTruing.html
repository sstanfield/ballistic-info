  • Set zero used for truing (generally use 100 meters)
  • Set truing range, should be transonic (mach 1.2) +/- 50, (transonic - 50) to (transonic+50)
  • Select elevation required to hit minimal sized target at range (either MIL or MAO)
  • Make adjustments to initial velocity or BC if required- for instance if using a chronograph then update velocity to accurate value and use the solved BC
  • Select the Solve button to solve for the BC with current velocity and velocity with current BC
  • If satisfied with result select:
    • "Save BC" to use the solved BC
    • "Save Velocity" to use the solved velocity

NOTE 1: You need to set the atmospheric conditions for when test is fired on atmospheric tab. These settings are used for both zero and range shots (i.e. zero rifle on same day as truing).

NOTE 2: If using a default weapon (not writable) then saving creates a new weapon as a copy of the current with updated info from truing. If using a non-default (writable) weapon then it will update the current weapon.
