BALLISTICS INFO:Pre-mission Zero Information
Page 1 Home Page 3

Users must enter zero information prior to a mission. This information is associated with the selected weapon. Each weapon has itâ€™s own zero information recorded.

Access this screen through selecting the navigation menu --> Zero.

  1. Select Zero Range.
  2. Atmospheric Settings can be entered if known. Entering atmospheric information will make calculations more accurate. NOTE: atmospheric information is based on conditions when weapon is zeroed. Atmospheric data includes: Altitude, Barometer (based on standardized pressure, this is NOT absolute pressure), Temp, and Humidity. Units of measurement may be adjusted on the right side of the unit symbol.
  3. Reset adjusts screen back to standard conditions.
