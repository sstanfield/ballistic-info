Atmospheric Conditions should be entered on location to obtain the most accurate firing solution. This is the same interface as when user enters Atmospheric Conditions when entering Pre-Mission Zero Information, however this information is based on field conditions.

  1. Turn On or Off atmospheric conditions.
  2. Adjust atmospheric conditions. This can only be input if atmospheric conditions are On. To change the units, touch the unit box.
  3. Reset clears all data entered.
